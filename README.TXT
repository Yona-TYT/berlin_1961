------------------------------------------------------------------------

Scenario : "Berlin 1961 - 'Simutrans-Player, tear down this wall!'"

------------------------------------------------------------------------


play with : Simutrans 120.0 (r7332),  pak 128 Version 2.5.0 (r1486)
mapsize   : 1024 * 1024, greater Berlin
options   : timeline on
            industrie_increase=0
            multiply_headquarter=100000
            factory_worker_radius=50
addon     : file: berlin_1961_v03.pak
            Berlin-Wall (big & small wall, fence, road, road_light,
                         search_light, tower, graffiti)
            border transit train station
            border police inspection vehicle
            normal fence for gameplay
extra     : berlin_1961.pdf for more information


To play this scenario pleace copy the PAK-File into the 'addons/pak128/'
directory (like C:/Users/[username]/Documents/Simutrans/addons/pak128/ )
or the different pak128 directory name. Now load simutrans with addons.
The scenario is also playable without this addon, but you won't get a
nice view of the Berlin wall, checkpoints and border troops.
